# Lista Rápida

## Descrição
Lista Rápida é um aplicativo simples desenvolvido em **React Native** para ajudar na organização de listas de compras. Com uma interface intuitiva, ele permite adicionar, marcar como comprado e remover itens com facilidade.

## Funcionalidades
- ✅ Adicionar itens à lista.
- ✅ Marcar itens como comprados.
- ✅ Remover itens da lista.
- 🎨 Interface responsiva e de fácil uso.

## 🛠 Tecnologias Utilizadas
- **React Native**: para desenvolvimento do aplicativo.
- **Expo**: para facilitar o ambiente de desenvolvimento.
- **React Icons (Ionicons)**: para uso de ícones.
- **useState**: para gerenciamento de estado.

## 🚀 Como Executar
1. Clone o repositório:
   ```sh
   git clone https://github.com/seu-usuario/lista-rapida.git
   ```
2. Acesse a pasta do projeto:
   ```sh
   cd lista-rapida
   ```
3. Instale as dependências:
   ```sh
   npm install
   ```
4. Inicie o projeto:
   ```sh
   npm start
   ```
5. Escaneie o QR Code no **Expo Go** para testar no celular.
